# Tkinter Calculator

A simple calculator built using **Python Tkinter GUI**.

## ✨ Features
- Perform basic arithmetic operations (+, -, ×, ÷).
- Clear & Reset functionality.
- User-friendly and responsive GUI.

## 🛠️ Requirements
- Python 3.x
- Tkinter (comes with Python by default)

## 🚀 How to Run
```bash
python calculator.py
